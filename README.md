# Doliver Finance
This project is for a personal study. The main goals is experimental the follow technologies:

- Spring Boot
- Docker
- Relational databases
- No-relational databases
- Messages and Event Streaming platforms

### How to run
Follow the steps

1. Build the project
```
./gradlew clean build
```
2. Execute Flyway Migration

3. Run application
   1. Java Application
   2. Docker
